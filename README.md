## üìñ Description

Welcome to **mcp-hello-world**, a simple example built in Kotlin to demonstrate the **Model Context Protocol (MCP)**. This client-server demo uses standard input and output (STDIO) for communication. The client launches an MCP server to run a simple 'greet' tool. This project is perfect for grasping the basics of MCP without dealing with complex large language models (LLMs).
